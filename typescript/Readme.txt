Comando Terminal : tsc app 
(sem necessidade de .ts) faz o trasnpiler do arquivo ts para o arquivo .js.

Comando Terminal : tsc -init 
cria o arquivo de configurações para o tsc
-> pode-se fazer uma series de configurações ao executar o transpiler, como dizer para qual versão do js 
será feito. Ex: ECMA 6
-> configuração -> outDir -> informar diretorio onde serão criados os arquivos js por padrão ele cria
junto com o arquivo origem

Comando Terminal : tsc -w
 Fazer o transpiler ficar assistindo as alterções dos arquivos td e faz transpiler para js 
em tempo real.

Os exemplos em si ensinam as novidades do Ecma 6, 
sobre o typeScript que é a possibilidade de tipagem de variaveis no javascript,
e sobre os conceitos de orientação a Objeto aplicados ao javascript.
